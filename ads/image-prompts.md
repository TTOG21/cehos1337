# Heuri-stica — Static Ad Image Prompts

Use these prompts with **Ideogram** (best for text-in-image) or **Flux Pro** (best for photorealistic).
Requested dimensions are listed per platform.

---

## Variant 1 — Pain Angle (Dark/Red)

**Ideogram prompt:**
```
A sleek dark UI dashboard (near-black #0D0D0D background) showing a UX heatmap with red zones highlighting problem areas. Bold white text overlay reads "Stop Losing Users to Bad UX". Small red accent logo block labeled "H" top-left. Modern sans-serif typography. Bottom CTA button in red: "Get a Free UX Audit". Clean, professional, high-contrast. 1080x1080px.
```

**Flux Pro prompt:**
```
Minimal dark tech aesthetic, almost black background. A laptop screen showing a UX heatmap interface with glowing red problem zones. No people. Clean flat composition. Accent color #FF3B3B. Professional B2B SaaS advertising style. Studio lighting effect. 1080x1080.
```

## Platform Dimensions Quick Reference

| Platform | Size | Use Variant |
|---|---|---|
| Meta Feed (square) | 1080×1080 | Any |
| Meta Stories / Reels | 1080×1920 | Pain or Contrarian |
| Google Display | 1200×628 | Outcome |
| LinkedIn Feed | 1200×627 | Identity or Outcome |
| TikTok In-Feed | 1080×1920 | Contrarian |

## Tools to Generate

1. **Ideogram** (best for text-in-image): [ideogram.ai](https://ideogram.ai)
2. **Flux Pro** (best photorealistic): [replicate.com/black-forest-labs/flux-pro](https://replicate.com/black-forest-labs/flux-pro) or [fal.ai](https://fal.ai)
3. **Nano Banana / Gemini Image** (Google): [aistudio.google.com](https://aistudio.google.com)

For text-heavy ads (headlines baked into image), use **Ideogram** — it has ~90% text accuracy vs ~30% for other tools.
